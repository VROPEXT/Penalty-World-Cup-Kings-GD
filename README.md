# Penalty Kings

GitHub Pages build for Penalty Kings.

The playable game entry point is `index.html`.

Recommended GitHub Pages settings:

- Repository name: `penalty-kings`
- Visibility: Public, unless your GitHub plan supports private Pages
- Pages source: Deploy from a branch
- Branch: `main`
- Folder: `/root`

After deployment, the game URL should look like:

`https://YOUR-USERNAME.github.io/penalty-kings/`
